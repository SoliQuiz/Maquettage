# Manifeste des Atomes

Ce fichier recense tous les atomes (composants unitaires) créés pour le projet SoliQuiz via Preline UI.

| Nom | Emplacement | Description | Variants |
|---|---|---|---|
| `heading-1` | `atoms/heading-1/` | Titre principal de page | - |
| `button-primary` | `atoms/button-primary/` | Bouton d'action principal | Solid, Outline |
| `input-text` | `atoms/input-text/` | Champ de saisie texte standard | Standard, Error |
| `label` | `atoms/label/` | Label pour formulaire | - |
| `alert-error` | `atoms/alert-error/` | Message d'erreur | Alert-soft |
